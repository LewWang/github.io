守破离

发表于 2017-05-07 | 分类于 哲学 | | 阅读次数

解释1：（日本剑道心诀）

​ 1.“守”：最初阶段须遵从老师教诲，认真练习基础，达到熟练的境界。

​ 2.“破”：基础熟练后，试着突破原有规范让自己得到更高层次的进化。

​ 3.“离”：在更高层次得到新的认识并总结，自创新招数另辟出新境界。

解释2：（正心馆的诠释）编辑

​ 守：无我即是守，放下自己的高傲，虚心学习周围所有人，三人行必有我师

​ 破：即摒弃糟粕，并非指招式技法上的，而是指内心中的怨恨、嫉妒、贪婪

​ 离：止即是离，适可而止，适时而止，能找到自己心愿停下脚步的人和地方

引自：百度百科-守破离词条

—————————————————————-2017.05.07 更———————————————————————–

  • 【守】这里用临摹或者模仿比较贴进我的理解；在这个阶段要做的就是寻找到自己的标杆参照物，可以是身边的你认为可以学习的人（三人行必有我师），也可以是此时学习方向的行业大牛；这个阶段关键是要沉住气，潜心打磨自己，接受外界传授的知识与经验，不断整合内化成自己的东西。这个阶段切记不要自作聪明的自己去改动。

    但是在【守】的阶段很容易进入“守旧”的黑洞，由于模仿过多形成了“依赖”。【守】的目的只在于借用前人的经验方法，达到熟能生巧。切记这个过程中不要被贴上了别人的标签。这个过程也要具有创新能力和自我主体意识。

  • 【破】待基础熟练之后，根据自身的经验及知识体系等向上进行突破改进，谋求更为精进的体系。但是如果在没有【守】好的前提下就盲目地【破】，很容易心浮气躁，没有扎实基础的支撑，得到的结果也会破绽百出，【破】的效果也就微乎其乎，甚至会引导你走向错误的方向。需要注意的是在【破】的阶段仍然需要【守】

  • 【离】自成一派，成为他人【守】的参照。不禁感叹天道往复，因果转圜啊

关于“守、破、离” 的哲学问题，由于涉世未深，目前理解力有限，今后会根据自身的感受继续更新。
